# PhaseFlow De Novo 序列设计分析报告

## 模型配置

- 模型：`outputs_set/output_set_flow32_missing15/best_model.pt`
- 架构：Set Encoder + Transfusion (Flow Matching + LM)
- 配置：flow_weight=32, lm_weight=1, dim=256, depth=6, heads=8
- 训练数据：missing ≤ 15（280,778 条）

## 1. 序列多样性分析

### 实验设置

- 数据源：missing_0.csv 中 Top 5% 强相分离相图（mean_pssi ≤ -0.730，509 条）
- 每个相图生成 100 条序列（temperature=1.0，无 top-k/top-p 限制）
- 总计生成 50,900 条序列

### 结果

| 指标 | Mean | Std | Median |
|------|------|-----|--------|
| 唯一率 | 1.000 | 0.000 | 1.000 |
| 平均编辑距离 | 16.945 | 0.241 | 16.986 |
| AA 组成方差 | 0.0027 | 0.0002 | 0.0027 |
| 平均序列长度 | 18.758 | 0.554 | 18.910 |
| 长度标准差 | 2.392 | 0.519 | 2.310 |
| RT Spearman | 0.447 | 0.214 | 0.488 |
| 预测相图 per-sample std | 0.257 | 0.020 | 0.254 |
| 预测相图跨样本方差 | 0.087 | 0.017 | 0.085 |

### 分析

- 唯一率 100%：每个相图 100 条序列全部不重复，LM 不是在死记硬背
- 编辑距离 ~17（序列长度 ~19）：生成序列之间差异很大，序列空间多样性充分
- RT Spearman 0.45：生成序列再预测回去与条件相图的相关性，接近模型直接预测能力（test set Spearman 0.41）
- 预测相图 std 0.257 vs 自然序列 target 0.76：方差坍缩问题存在，但不影响序列多样性

结论：序列空间多样性好，功能空间受限于 Flow 预测端的方差坍缩。

## 2. Log-Likelihood 区分度验证

### 实验设置

- 数据：test_set.csv（500 条，无缺失值）
- 强/弱划分：按 mean_pssi 排序取极端子集

### 实验一：三组总体对比

| 组别 | Mean LL |
|------|---------|
| 自然序列 + 正确相图 | -2.174 |
| 自然序列 + 错误相图（随机打乱） | -2.216 |
| 随机序列 + 任意相图 | -2.424 |

- 自然 vs 随机序列差异：0.250（LM 能区分序列合理性）
- 正确 vs 打乱相图差异：0.042（条件依赖弱）

### 实验二：极端强/弱交叉配对

**Top 50 极端（强 mean_pssi=-0.775，弱 mean_pssi=+0.445，差 1.22）：**

| 组别 | Mean LL |
|------|---------|
| 强序列 + 强相图（正确） | -2.222 |
| 强序列 + 弱相图（错误） | -2.342 |
| 弱序列 + 弱相图（正确） | -2.123 |
| 弱序列 + 强相图（错误） | -2.248 |

- 强序列：正确 - 错误 = +0.120
- 弱序列：正确 - 错误 = +0.125

**Top 125（差 0.85）：** 正确 - 错误 = +0.056 / +0.084

### 实验三：广义相图二分类

| 划分 | 强弱 PSSI 差 | 分类准确率 |
|------|-------------|-----------|
| Top 50 | 1.22 | 89.0% |
| Top 125 | 0.85 | 77.6% |
| Top 250 | 0.52 | 67.8% |

### 分析

- LM 主要信号来自序列无条件分布（~75%），条件依赖较弱（~25%）
- LL 区分度与相图差距成正比：极端情况下准确率 89%，中等差距下降到 68%
- LL 适合做序列合理性过滤器，不适合做条件匹配打分
- 联合排序应以 Flow Spearman 为主，LL 辅助

## 3. De Novo 序列设计管线

### 管线流程

```
Step 1: 选择目标相图 → missing_0 Top 5% (509条)
Step 2: 批量生成 → 每个相图 LM 生成 20 条 → 10,180 条候选
Step 3: 生物学合理性过滤 → 长度≥10 + 合法AA + 无低复杂度 + 去重 + 新颖性
Step 4: 输出
```

### 参数配置

| 参数 | 值 |
|------|-----|
| 目标相图来源 | missing_0.csv Top 5% (mean_pssi ≥ 0.432) |
| 目标相图数 | 509 |
| 每相图候选数 | 20 |
| 生成温度 | 1.0 |
| 最大序列长度 | 25 |

### 生物学过滤规则

仅检查生物学合理性，不过滤相分离特征（GRAVY、芳香性、电荷等由模型分布决定）：

- 长度 ≥ 10
- 仅含 20 种标准氨基酸
- 单 AA 占比 ≤ 40%
- 无连续 4 个相同 AA
- 序列去重
- 训练集新颖性检查

### 漏斗统计

| 步骤 | 数量 | 通过率 |
|------|------|--------|
| 目标相图 | 509 | — |
| 生成候选 | 10,180 | — |
| 生物学过滤 + 去重 + 新颖性 | 8,115 | 79.7% |

- 过滤淘汰 2,065 条（20.3%），主要原因：序列过短（<10 AA）
- 去重后 0 条重复（100% 唯一）
- 新颖性检查 0 条与训练集重叠

### 结果

| 指标 | 值 |
|------|-----|
| 输出序列数 | 8,115 |
| 覆盖目标相图数 | 509/509 (100%) |
| 每相图平均候选数 | 15.9 |
| 序列长度 mean | 17.4 |
| 序列长度 median | 20 |

### 分析

- 79.7% 生物学过滤通过率，模型生成的序列大部分是合理的蛋白质片段
- 509 个目标相图全覆盖，每个相图平均 ~16 条有效候选
- 序列长度分布稳定（mean 17.4，median 20），与自然序列分布一致
- 100% 唯一 + 0 训练集重叠，模型不是在背训练数据

### 输出文件

- `analysis/de novo/de_novo_results_v2/candidates.csv` — 8,115 条全量
- `analysis/de novo/de_novo_pipeline.py` — 管线脚本

## 4. Inference-Time Optimization (ITO)

### 方法

对每个 Top 5% 相图生成 200 个候选序列，经生物学过滤后用 Flow 反向预测相图（generate_phase），计算 Spearman(预测, 目标)，每个相图取 top-1。

### 参数配置

| 参数 | 值 |
|------|-----|
| 目标相图 | missing_0.csv Top 5% (509条) |
| 每相图候选数 | 200 |
| 生物学过滤 | 同 Section 3 |
| 打分方法 | Flow generate_phase → Spearman |
| 选择策略 | 每相图 top-1 |

### 结果

| 指标 | 值 |
|------|-----|
| 目标相图数 | 509 |
| 成功 | 509 (100%) |
| 每相图有效候选 (过滤后) | ~150 条 |

**Best Spearman (top-1 per phase)：**

| 统计量 | 值 |
|--------|-----|
| Mean | 0.5672 |
| Median | 0.5735 |
| Min | -0.0177 |
| Max | 0.9441 |

**Mean Spearman (200 候选平均)：** 0.2663

### 分析

- ITO 将平均 Spearman 从裸生成的 0.266 提升到 0.567，best-of-200 筛选增益显著
- 分布集中在 0.4–0.8 区间，大部分相图能找到匹配度较好的序列
- 1 条 Spearman < 0（source_data_idx=67），该相图有 3 个位置达到 PSSI 最大值 1.64，std=0.72，模型预测 std 仅 0.34，属于方差坍缩导致的极端 case
- 方差坍缩仍是主要瓶颈：目标相图高波动时，模型无法重建，ITO 也无法弥补

### 输出文件

- `analysis/de novo/de_novo_ito_results/ito_top1.csv` — 509 条 top-1 序列
- `analysis/de novo/de_novo_ito_results/spearman_hist.png` — Spearman 分布直方图
- `analysis/de novo/de_novo_ito.py` — ITO 脚本

## 5. 分布对比分析：LLPS 关键驱动特征

### 对比三组

1. De Novo ITO (509 条) — best-of-200 筛选
2. De Novo Gen (8,115 条) — 直接生成 + 生物学过滤
3. GT Top5% (509 条) — 原始实验序列

### LLPS 关键特征对比

| 特征 | De Novo ITO | De Novo Gen | GT Top5% | KL(ITO\|\|GT) | KL(Gen\|\|GT) |
|------|-------------|-------------|----------|--------------|--------------|
| gravy | 0.0321 | -0.0789 | 0.0084 | 0.164 | 0.124 |
| hydrophobic_frac | 0.4515 | 0.4390 | 0.4493 | 0.302 | 0.230 |
| aromatic_frac | 0.1064 | 0.0970 | 0.1028 | 0.785 | 0.363 |
| charged_frac | 0.1512 | 0.1617 | 0.1648 | 0.262 | 0.181 |
| net_charge_per_res | 0.0271 | 0.0260 | 0.0200 | 0.484 | 0.352 |
| polar_frac | 0.2477 | 0.2518 | 0.2425 | 0.343 | 0.645 |
| F_frac | 0.0557 | 0.0485 | 0.0525 | 0.568 | 0.219 |
| W_frac | 0.0192 | 0.0181 | 0.0171 | 0.399 | 0.164 |
| Y_frac | 0.0315 | 0.0304 | 0.0333 | 0.617 | 0.492 |
| length | 17.86 | 17.42 | 14.14 | 0.408 | 0.373 |

### 分析

- Gen 在多数特征上 KL 更小，分布更接近 GT；ITO 的 Spearman 筛选引入了一定分布偏移
- ITO 在芳香族 (KL=0.785) 和 F_frac (KL=0.568) 上偏移最大，筛选倾向选中芳香族偏多的序列
- Gen 在极性 (KL=0.645) 上偏移较大
- 疏水性 (GRAVY, hydrophobic_frac) 三组均值接近，对齐较好
- 长度是最大的系统性差异：生成序列集中在 20 AA（ITO 62%、Gen 57%），GT 分布均匀 (5-20)，mean=14.1
- 均值层面三组差异不大，主要差异在分布形状上
